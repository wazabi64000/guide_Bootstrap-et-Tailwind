Classes utilitaires et personnalisation

1. Bootstrap : utilitaires CSS

Bootstrap propose de nombreuses classes utilitaires pour :

  • **Espacement :** m-3, p-2
  • **Couleurs :** bg-primary, text-success
  • **Alignement :** text-center, d-flex justify-content-between

Exemple :

<div class="p-3 mb-3 bg-light text-center">Texte centré</div>

2. Tailwind : utilitaires CSS

Tailwind fonctionne uniquement avec des classes utilitaires :

  • **Espacement :** m-4, p-2
  • **Couleurs :** bg-blue-500, text-white
  • **Flex et Grid :** flex justify-between, grid grid-cols-3 gap-4
  • **Responsive :** sm:, md:, lg:

Exemple :

<div class="p-4 mb-4 bg-gray-200 text-center sm:text-left">Texte responsive</div>

3. Personnalisation

Bootstrap :

  • Modifier les variables Sass pour couleurs, marges, typographie.
  • Créer un thème personnalisé.

Tailwind :

  • Configurer tailwind.config.js pour ajouter des couleurs, spacing, polices.
  • Créer des composants réutilisables avec des utilitaires combinés.
// Exemple Tailwind config
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: '#1da1f2',
      },
      spacing: {
        72: '18rem',
      }
    }
  }
}
