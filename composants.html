Composants Bootstrap & Tailwind

1. Boutons

Exemple Bootstrap :

<button class="btn btn-primary">Bootstrap Primary</button>
<button class="btn btn-success">Bootstrap Success</button>

Exemple Tailwind :

<button class="bg-blue-600 text-white px-4 py-2 rounded">Tailwind Blue</button>
<button class="bg-green-500 text-white px-4 py-2 rounded">Tailwind Green</button>

2. Cartes / Cards

Bootstrap :

<div class="card" style="width: 18rem;">
  <img src="assets/images/sample.jpg" class="card-img-top" alt="Image">
  <div class="card-body">
    <h5 class="card-title">Titre</h5>
    <p class="card-text">Description de la carte.</p>
    <a href="#" class="btn btn-primary">Action</a>
  </div>
</div>

Tailwind :

<div class="bg-white shadow rounded p-4 w-72">
  <img src="assets/images/sample.jpg" class="rounded mb-3">
  <h5 class="text-lg font-bold mb-2">Titre</h5>
  <p class="mb-3">Description de la carte.</p>
  <button class="bg-blue-600 text-white px-4 py-2 rounded">Action</button>
</div>

3. Navbar / Barre de navigation

Bootstrap :

<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Logo</a>
  <div class="collapse navbar-collapse">
    <ul class="navbar-nav">
      <li class="nav-item"><a class="nav-link" href="#">Accueil</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Contact</a></li>
    </ul>
  </div>
</nav>

Tailwind :

<nav class="flex items-center justify-between bg-gray-200 p-4">
  <a href="#" class="font-bold text-lg">Logo</a>
  <div>
    <a href="#" class="mr-4">Accueil</a>
    <a href="#">Contact</a>
  </div>
</nav>

4. Formulaires

Bootstrap :

<form>
  <div class="mb-3">
    <label for="email" class="form-label">Email</label>
    <input type="email" class="form-control" id="email">
  </div>
  <button type="submit" class="btn btn-primary">Envoyer</button>
</form>

Tailwind :

<form class="space-y-4">
  <div>
    <label for="email" class="block mb-1">Email</label>
    <input type="email" class="border border-gray-300 p-2 rounded w-full">
  </div>
  <button class="bg-blue-600 text-white px-4 py-2 rounded">Envoyer</button>
</form>

Résumé composants

  • Bootstrap : composants préconçus, classes faciles à utiliser.
  • Tailwind : utilitaires pour créer des composants personnalisés.
  • Les deux permettent de construire rapidement des interfaces modernes.
