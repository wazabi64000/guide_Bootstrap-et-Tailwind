Installation de Bootstrap et Tailwind CSS

1. Installer Bootstrap

Vous pouvez utiliser Bootstrap de deux manières : via CDN ou npm.

a) Utilisation via CDN :

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

b) Installation via npm :

npm install bootstrap
// Puis inclure dans votre fichier JS principal
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';

2. Installer Tailwind CSS

Vous pouvez également utiliser Tailwind via CDN ou l’installer via npm pour personnaliser la configuration.

a) Utilisation via CDN :

<script src="https://cdn.tailwindcss.com"></script>

b) Installation via npm (avec PostCSS) :

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init
// Ajouter Tailwind dans votre CSS
@tailwind base;
@tailwind components;
@tailwind utilities;

3. Exemple simple combinant Bootstrap et Tailwind

<div class="container my-5">
  <h1 class="text-center text-blue-600">Bonjour Bootstrap & Tailwind</h1>
  <button class="btn btn-primary mt-3">Bootstrap Button</button>
  <button class="bg-green-500 text-white px-4 py-2 rounded mt-3">Tailwind Button</button>
</div>

Résumé installation

  • Bootstrap : CDN rapide ou npm pour projet complet.
  • Tailwind : CDN simple ou npm avec configuration pour personnalisation.
  • Les deux peuvent être combinés pour expérimenter.
